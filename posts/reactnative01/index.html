App.js 代码如下：

import { StatusBar } from "expo-status-bar";
import { StyleSheet, Text, View } from "react-native";

export default function App() {
    return (
        <View style={styles.container}>
            <Text>Open up App.js to start working on your app!</Text>
            <StatusBar style="auto" />
        </View>
    );
}

const styles = StyleSheet.create({
    container: {
        flex: 1,
        backgroundColor: "#fff",
        alignItems: "center",
        justifyContent: "center",
    },
});

从上面代码可以看出， 在 RN 中样式处理类似于 React 项目的 Styles Component 风格， 并且 RN 提供了一些原生组件来适配移动端项目：

View 类似于 Web 中的 div, 默认就是 flex 布局

Text 类似于 Web 中的 p, 在移动端项目不能直接使用文字，比如下面这种形式会报错：

<View>some text...</View

所有的文字必须要包裹在 Text标签中

3.1.1 Input 组件

首先明确下 Input 组件的 Props, 如下所示：

interface InputProps {
    label: string;
    value: string;
    onChangeText: (value: string) => void; // 输入文字后的回调
    onsubmitEditing: (value: string) => void;
    placeholder: string;
    resetProps: any[]; // TextInput组件 Props
}

在 RN 中使用 TextInput处理文本输入它有一个名为 onChangeText 的属性，此属性接受一个函数，而此函数会在文本变化时被调用。另外还有一个名为 onSubmitEditing 的属性，会在文本被提交后（用户按下软键盘上的提交键）调用。

使用 TextInput组件进行二次封装， 代码如下：

import { useState } from "react";
import { StyleSheet, TextInput, View, Text } from "react-native";

export default function Input({ label, value, placeholder, onChangeText, style, ...resetProps }) {
    const [isFocused, setIsFocused] = useState(false);
    return (
        <View style={[styles.container, style]}>
            {label && (
                <View style={styles.labelContainer}>
                    <Text style={styles.label}>{label}</Text>
                </View>
            )}
            <TextInput
                style={[styles.input, isFocused && styles.inputFocused]}
                value={value}
                placeholder={placeholder}
                onChangeText={onChangeText}
                onFocus={() => setIsFocused(true)}
                onBlur={() => setIsFocused(false)}
                {...resetProps}
            />
        </View>
    );
}

const styles = StyleSheet.create({
    labelContainer: {
        marginBottom: 16,
    },
    label: {
        fontSize: 14,
        color: "#000",
    },
    input: {
        height: 40,
        borderColor: "#d9d9d9",
        borderWidth: 1,
        borderRadius: 4,
        paddingHorizontal: 8,
        backgroundColor: "#fff",
        fontSize: 16,
    },
    inputFocused: {
        borderColor: "#1890ff",
        borderWidth: 2,
    },
});

效果如下所示：

但是这里存在一个问题， 当键盘升起的时候， 键盘可能会遮挡住 输入框， 因此可以使用 RN 提供的KeyboardAvoidingView来处理

本组件用于解决一个常见的尴尬问题：手机上弹出的键盘常常会挡住当前的视图。本组件可以自动根据键盘的高度，调整自身的 height 或底部的 padding，以避免被遮挡。

调整后的代码如下：

<KeyboardAvoidingView style={[styles.container, style]}>
    {label && (
        <View style={styles.labelContainer}>
            <Text style={styles.label}>{label}</Text>
        </View>
    )}
    <TextInput
        style={[styles.input, isFocused && styles.inputFocused]}
        value={value}
        placeholder={placeholder}
        onChangeText={onChangeText}
        onFocus={() => setIsFocused(true)}
        onBlur={() => setIsFocused(false)}
        {...resetProps}
    />
</KeyboardAvoidingView>

3.1.2 Button 组件封装

RN 提供了原生的 Button 组件， 但是不符合样式 需要使用Pressable组件进行封装。 官网说明如下：

这个组件的样式是固定的。所以如果它的外观并不怎么搭配你的设计，那你需要使用 Pressable 组件来定制自己所需要的按钮，或者你也可以在 github.com 网站上搜索 ‘react native button’ 来看看社区其他人的作品。

Pressable 官网说明如下：

Pressable 是一个核心组件的封装，它可以检测到任意子组件的不同阶段的按压交互情况

封装代码如下：

import { Pressable, StyleSheet, Text } from "react-native";

export const BUTTON_TYPE_CONFIG = {
    PRIMARY: "primary",
    DANGER: "danger",
};

export default function Button({ text, onPress, type = "primary", style, ...resetProps }) {
    return (
        <Pressable
            onPress={onPress}
            style={({ pressed }) => [styles.button, styles[type], pressed && styles.buttonPressed, style]}
            {...resetProps}
        >
            <Text style={[styles.text, styles[`${type}Text`]]}>{text}</Text>
        </Pressable>
    );
}

const styles = StyleSheet.create({
    button: {
        height: 40,
        borderRadius: 4,
        justifyContent: "center",
        alignItems: "center",
        paddingHorizontal: 16,
    },
    primary: {
        backgroundColor: "#1890ff",
    },
    default: {
        backgroundColor: "#f5f5f5",
        borderColor: "#d9d9d9",
        borderWidth: 1,
    },
    danger: {
        backgroundColor: "#ff4d4f",
    },
    buttonPressed: {
        opacity: 0.75,
    },
    text: {
        fontSize: 16,
    },
    primaryText: {
        color: "#fff",
    },
    defaultText: {
        color: "#000",
    },
    dangerText: {
        color: "#fff",
    },
});

这里的 Style 参考了官方文档的说明，接受一个函数

可使用普通视图样式，或者一个函数来根据按压状态布尔值返回视图样式

3.2 TodoList 业务开发

首先是Header组件， Header 组件由一个输入框和一个按钮组成， 用户输入代办事项， 点击按钮进行添加， 代码如下所示：

import { useState } from "react";
import { StyleSheet, View } from "react-native";
import Input from "../ui/Input";
import Button from "../ui/Button";

export default function Header({ onAdd }) {
    const [content, setContent] = useState("");
    return (
        <View style={styles.container}>
            <Input placeholder="请输入要添加的事项..." style={styles.input} value={content} onChangeText={setContent} />
            <Button text="添加" onPress={onAdd} />
        </View>
    );
}

const styles = StyleSheet.create({
    container: {
        height: 40,
        flexDirection: "row",
        gap: 10,
        marginTop: 20,
    },

    input: {
        flex: 1,
    },
});

然后需要封装 TodoItem 组件：

import { View, Text, StyleSheet } from "react-native";
import Button from "../ui/Button";

export default function TodoItem({ content, id, completed, onCompleted, onDelete }) {
    return (
        <View style={styles.container}>
            <Text style={[{ fontSize: 16 }, completed && styles.textCompleted]}>{content}</Text>

新建 TodoList 组件， 代码如下：

import { useCallback, useState } from "react";
import { FlatList, StyleSheet, Text, View } from "react-native";
import Header from "./Header";
import TodoItem from "./TodoItem";

export default function TodoList() {
    const [todoList, setTodoList] = useState([]);

    const onAdd = useCallback(
        (content) => {
            setTodoList((prev) => [
                ...prev,
                {
                    id: Math.ceil(Math.random() * 1000),
                    content,
                    completed: false,
                },
            ]);
        },
        [todoList]
    );

    const onCompleted = useCallback(
        (id) => {
            setTodoList((prev) =>
                prev.map((item) => (item.id === id ? { ...item, completed: !item.completed } : item))
            );
        },
        [todoList]
    );

    const onDelete = useCallback(
        (id) => {
            setTodoList((prev) => prev.filter((item) => item.id !== id));
        },
        [todoList]
    );
    return (
        <View style={styles.container}>
            <Header onAdd={onAdd} />
            <FlatList
                style={styles.list}
                data={todoList}
                renderItem={({ item }) => <TodoItem {...item} onCompleted={onCompleted} onDelete={onDelete} />}
            />
        </View>
    );
}

const styles = StyleSheet.create({
    container: {
        flex: 1,
        marginHorizontal: 20,
    },

    list: {
        flex: 1,
    },
});
