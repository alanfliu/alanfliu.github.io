基于NestJS + JWT实现登陆注册及权限控制

Jun 29, 2024
1 min read

1 需求背景

最近在学习 NestJS 和后端相关知识， 想做个简单的管理系统来练练手， 想来想去就从学生选课系统入手。

首先， 用户分为三种用户： 学生、老师和管理员， 各自有不同的功能。

学生和老师可以注册，注册的时候会发邮件来验证身份， 注册后就可以登陆系统。管理员是内置的，不需要注册。

学生可以查看课程成绩， 报名课程并查看课程进度， 最后通过考试， 获得课程证书。

老师可以发布课程， 给对应课程名下的学生的成绩进行打分， 并公布学生成绩。

管理员需要审核老师发布的课程， 审核通过的课程才能被学生看到。管理元也可以冻结角色为学生和老师的账户， 被冻结的账户不能登陆系统

2 技术选型

用到的技术如下所示：

  1. 后端： NestJS + TypeOrm + Typescript
  2. 前端： React + Redux + React Router + Typescript + TailwindCSS
  3. 数据库：Mysql
  4. 缓存： Redis
  5. 部署： Docker + Docker Compose + Nginx

需求分析

在学生选课系统中， 最基础的就是用户管理模块， 也就是登陆注册。

首先是注册功能实现， 需要以下接口：
